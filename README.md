# Leeghwaterplas_Interactive_Map
In this project I will work on an interactive map for the location Leeghwaterplas in Almere, The Netherlands.
This is to showcase our ideas for the given location! 
Stichting Stad & Natuur Almere has given us the opportunity to work on this project.

## Features
- Interactive map of Leeghwaterplas
- Labels/Titles for important locations
- Detailed information for interactable objects
- User-friendly interface
